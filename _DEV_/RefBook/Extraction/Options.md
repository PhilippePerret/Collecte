# Options d'extraction

Ce fichier contient la description des options qu'on peut passer à la méthode `Collecte::Extractor#extract_data` pour spécifier la sortie qu'on veut obtenir.

~~~

  :format
      Le format de sortie, parmi :html, :text, :xml.

  :path
      Le path éventuel du fichier produit. Par défaut, ce
      sera.

  :open_file
      Si true, le fichier produit sera ouvert à la fin de sa
      construction.

~~~
